About Us

Welcome to my Full Stack Java app

This is my first Full Stack Java e-commerce site or shopping site. I have used Java before for smaller projects as it is the first programming language I learned in depth. I like most people don't really count HTML and CSS as programming languages. I also only knew enough JavaScript to help with making my web pages more interactive. I later learned Full Stack Web Development which was using JavaScript on the Back End which was Node.js.

This is first time I've used Thymeleaf which is a server-side Java template engine for web and standalone environments. After learning JavaScript Full Stack Development I am excited at the prospect of using Java for Full Stack. This project is using the Java Back end to create the server, handle page routing and create the PostgreSQL database.

I am also using bootstrap to aid in quicker styling of the web pages. While also using HTML and CSS.
